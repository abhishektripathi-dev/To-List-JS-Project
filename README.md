## How to Use

1. Open `index.html` in your web browser
2. Type a task in the input field
3. Click the **Add** button to add the task
4. Click on a task to mark it as complete (strikethrough effect)
5. Click the **Ã—** button on a task to delete it

## Technologies Used

- **HTML5**: Structure and semantic markup
- **CSS3**: Styling with flexbox and gradients
- **JavaScript (ES6)**: Event handling and DOM manipulation
- **Local Storage API**: Persistent data storage

## Local Storage

Tasks are automatically saved to the browser's local storage whenever you:
- Add a new task
- Mark a task as complete/incomplete
- Delete a task

Data persists even after closing and reopening the browser.
